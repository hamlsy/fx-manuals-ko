# SONICAKE Matribox Summary

## 1) 개요
- 제품 포지션: 컴팩트 멀티이펙트 프로세서
- 펌웨어 기준: v1.0.4
- 추천 사용자: 일렉기타/어쿠스틱기타/베이스/키보드 사용자

## 2) 주요 기능/특징
- 130개+ 이펙트, 40개+ 앰프 모델
- 9개 동시 이펙트 체인 구성(모듈 이동 가능)
- 루퍼 + 드럼머신 + USB 오디오 인터페이스 기능

## 3) 주요 스펙
- A/D/A: 24-bit
- 샘플레이트: 44.1 kHz
- SNR: 110 dB
- 프리셋: 사용자 99 / 팩토리 99
- 루퍼: 최대 90초(Pre 모드), Post 모드 최대 45초
- 드럼머신: 100 패턴
- 입출력: 1/4" 입력, 1/4" L/R 출력, 1/8" AUX/헤드폰, USB-C
- 전원: DC 9V 센터 마이너스 500mA
- 크기/무게: 200 x 137.5 x 53.6 mm / 732 g

## 4) 대략 시세
- 신규(출시가/현재가): 확인 필요
- 중고 시세 범위: 확인 필요
- 가격 메모: 지역/상태/구성품에 따라 변동

## 5) 후기 요약
### 공통 장점
- 입문~중급 사용자가 접근하기 쉬운 올인원 구성
- 다양한 앰프/캐비넷/공간계 효과를 한 기기에서 처리 가능
- USB 인터페이스와 전용 소프트웨어 지원

### 공통 단점
- 고가 플래그십 대비 세밀한 톤 튜닝 한계 체감 가능
- 패치 정리를 하지 않으면 운용 복잡도가 빠르게 올라감

### 체감 포인트
- 스톰프 모드와 프리셋 모드를 병행하면 라이브 활용도가 높음
- 노이즈 환경에서는 NR/Gate 세팅 최적화가 중요

## 6) 한 줄 평
- 가격 대비 기능 밀도가 높은 실전형 보급 멀티이펙터.
