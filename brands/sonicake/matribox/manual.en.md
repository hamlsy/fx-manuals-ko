## 1) Attention
### Handling
- Keep the unit dry. If liquid is spilled, power off immediately.
- Do not block ventilation openings.
- Keep away from heat sources.
- Disconnect during storms.
- Avoid use in strong electromagnetic fields.

### Power and I/O Connections
- Always power off this unit and connected equipment before connecting/disconnecting cables.
- Disconnect all cables and adapter before moving the unit.

### Cleaning
- Clean with a dry cloth only.

### Alterations
- Do not open the unit.
- Do not service it yourself.
- Opening the chassis voids warranty.

### AC Adapter
- Use only `DC 9V center-negative 500mA`.
- Wrong adapters may cause malfunction/safety hazards.
- Use a properly rated AC outlet.
- Unplug during lightning storms or long non-use.

### Malfunction Procedure
1. Disconnect AC adapter and power off.
2. Disconnect all other cables.
3. Prepare model/serial/symptom details.
4. Contact SONICAKE support.

## 2) Overview
Matribox is a compact multi-effects processor with 130+ effects for electric guitar, acoustic guitar, bass, keyboard, and more.

Highlights:
- 40+ legendary amp models
- 9 movable effect modules (effect matrix)
- White-box digital modeling approach for realistic response

## 3) Panel Introduction
### Top Panel Controls
1. Master volume
2. Screen (preset and operation info)
3. `PARA` knob: rotate/press for parameter and menu operations
4. `GLOBAL` button
5. `DRUM` button (press: on/off, hold: drum menu)
6. `BACK` button (previous menu)
7. `SAVE` button (save/rename/copy preset)
8. Bottom knobs (menu-dependent parameter controls)
9. Footswitches:
- preset up/down
- press both: tuner
- hold both: looper
10. Expression pedal:
- controls assigned parameter/volume
- toe switch toggles pedal function

### Rear/Side I/O
11. 1/4" TS mono instrument input
12. 2 x 1/4" TS stereo outputs (L/R)
13. 1/8" TRS stereo aux input
14. 1/8" TRS stereo headphone output
15. USB Type-C (editor software + audio interface)
16. DC 9V center-negative power input

## 4) Main Menu
Main screen fields:
- Preset number: `P01-P99` (user), `F01-F99` (factory)
- Preset state: indicates unsaved edits
- Drum state: on/off indication
- EXP state: on/off indication
- Preset info display area
- BPM: `40-250`
- Mode: `Preset` or `Stomp`
- P-VOL: preset volume

## 5) Working Modes
Matribox has two modes with different footswitch behavior.

### Preset Mode
- Press a footswitch to switch presets.

### Stomp Mode
- Press a footswitch to toggle assigned module on/off.
- Module assignment is editable in Effects Editing.

### Mode Switching
- In Main Menu, hold the designated footswitch to toggle Preset/Stomp.

### Tap Tempo
- Hold designated footswitch to enter Tap Tempo.
- Tap to set preset BPM (screen flashes current BPM).
- Hold again to exit.

## 6) Effects Editing Menu
- From Main Menu, press `PARA` to enter.
- Rotate `PARA` to choose module.
- Press `PARA` to move parameter page.
- Three bottom knobs map to on-screen parameters.
- Use for module on/off, effect selection, parameter edits.
- In Stomp Mode, `FS Switch` appears to assign footswitch control to current module.
- Hold `PARA` in this menu to adjust effect chain order.

## 7) Save / Tuner / Looper / Drum
### Save
Press `SAVE` to store edits (parameters/control info, etc.), choose destination preset, and rename preset.

### Tuner
- Press two footswitches together to enter tuner.
- Tuner modes: `Mute`, `Bypass`, `Thru`.
- Calibration range: `435Hz-445Hz`.

### Looper
- Hold both footswitches to enter looper.
- Record/play/overdub via footswitch sequence.
- Stop all with stop footswitch; hold to clear all.
- Preset can be changed with `PARA` knob.

Looper modes:
- `Pre`: mono dry recording, up to 90 sec
- `Post`: stereo wet recording, up to 45 sec

### Drum
- Hold `DRUM` button to open drum menu.
- Set rhythm style, speed, and volume.
- When `Sync` is on, drum tempo follows preset BPM.

## 8) Global Settings
Enter with `GLOBAL` button. Rotate/press `PARA` to select items/pages.

### Input/Output
- Input Level: `-20dB to +20dB`
- No CAB (L/R): bypass CAB for selected output channel regardless of preset

### USB Audio
- Rec Level: `-20dB to +20dB`
- Monitor Level: `-20dB to +6dB`
- Rec Mode (L/R): choose dry or wet signal for recording channel

### Tap Tempo Mode
- Can force-sync `FX1`, `FX2`, `DLY`, `MOD` to preset BPM

### EXP Setting
- Target: choose controlled target/parameter
- EXP Range: set min/max pedal range
- VOL Range: set volume range and effective pedal zone
- Calibrate: run pedal calibration routine

### Factory Reset
- Restores factory state and erases user data

### About
- Shows firmware information

## 11) Troubleshooting
### Device Won't Turn On
- Check power connection.
- Verify adapter operation.
- Confirm correct adapter spec (`DC 9V center-negative 500mA`).

### No Sound / Low Sound
- Check cable connections.
- Check output volume and module volume.
- If EXP controls volume, verify pedal position/range.
- Check preset volume and input mute state.

### Noise
- Check cables and instrument output jack.
- Confirm correct power adapter.
- Use NR module when source instrument is noisy.

### Sound Problems
- Check all cable connections and connected pedals.
- Verify effects settings are not at extreme values.

### Expression Pedal Problems
- Check EXP on/off settings.
- Run calibration.
